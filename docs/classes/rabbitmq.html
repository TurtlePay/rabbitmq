requestReply

  • requestReply<T, U>(queue: string, payload: T, timeout?: number, useOneTimeQueue?: boolean): Promise<U>

  • Sends a message to the given queue requesting that we receive a reply from the worker that handles the message. This method will wait for the worker to complete the request and the reply to be received before the promise will resolve

    Type parameters

    • T

    • U

    Parameters

    • queue: string

      the name of the queue to send our message to

    • payload: T

      the payload to be sent with the message

    • timeout: number = 5000

      the amount of time (ms) that we are willing to wait for a reply

    • useOneTimeQueue: boolean = false

      whether we should use a one-time use queue to receive the reply (this may be necessary in some use cases)

    Returns Promise<U>
